## Set up guide
1. Install dependencies `npm i`
2. Add your SMTP creds into `.env`
```js
SMTP_LOGIN=*login*
SMTP_PASSWORD=*password*
SMTP_PORT=*port*
SMTP_SERVER=*server*
```
3. Change sender and email template in your `mailer.ts`
4. To run the application simply use `npm start /path/to/csv`
